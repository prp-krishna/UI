Calculate distance travelled by light

Time (in years) :
km


Example :
Distance traveled by light in 1 Year = 94.6x1012 km

Explained below :

Speed of Light = 3x108m/s
OR 3x106km/sec

Time required for light to cover 94.6x1012km (obtained by speed of light x total seconds in 1 year) = 1 Year.

We have formula,

Speed = Distance / Time

Similarly,
Distance = Speed x Time and

Time = Distance / Speed

For the concept of light, we can use

c = ly / year

For 1 Light Year (ly)

1ly = c x 1year

1ly = 94.6x1012km
